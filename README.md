### Random Forest Classifier

* Random Forest Classifier is a supervised machine learning algorithm used for classification and prediction tasks. It works by creating multiple Decision Trees during training and combines their outputs to make a final prediction.
#### How it Works

*   Select random samples from the dataset.

*   Build multiple decision trees using these samples.
*   Each tree independently predicts an output.
*   The final prediction is determined by majority voting among all trees.
